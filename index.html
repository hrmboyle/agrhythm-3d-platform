AgRhythm Platform

Interactive 3D visualization of website structure

Nodes represent different pages and components. Lines show connections and data flow.

Controls:

🖱️ Left Click + Drag: Rotate

🖱️ Right Click + Drag: Pan

🖱️ Scroll: Zoom

🖱️ Hover: Highlight connections

🖱️ Click Node: Show details

Edit Mode:

🖘️ Click Node: Edit properties

➕ Add Node Button: Create new node

🖘️ Double-click Space: Create node at cursor

🖘️ Shift + Click Node: Start connection

🖘️ Right-click: Cancel connection
